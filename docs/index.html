Key Features

  • Interactive Transcription: Transcribe speech directly into the editor using your microphone.
  • AI-Powered Polishing: Correct grammar, improve phrasing, and fix typos with Gemini or local AI models.
  • Modern Theming: Choose between beautiful light and dark themes.
  • Flexible AI Options: Switch between Google's Gemini API and local AI models.
  • Session Management: Save, create new, or open past transcription sessions.
  • Persistent Settings: Your preferences (API key, AI service, theme) are saved.

Using the AI Polishing Feature

The transcription feature is always free and works without any API key.

To use the AI-powered 'Polish' feature, you'll need an API key from Google AI Studio for the Gemini model. You can usually get a free tier.

Watch this video to learn how to get your key: How To Get Your FREE Google Gemini API Key (2025).

Alternatively, if you have a powerful enough machine, you can set up a local AI model (e.g., via LM Studio) and use it for polishing without needing a cloud API key.
